# ぽんぽんカレンダー

シンプルな習慣化用カレンダーWebアプリです。  
日付をクリックするとスタンプが「ぽん」と付き、状態は端末の localStorage に保存されます。

## 機能
- 月表示カレンダー
- 日付クリックでスタンプON / OFF
- 月ごとの状態を localStorage に保存
- 前月・翌月切り替え
- レスポンシブ対応（PC / スマホ）
